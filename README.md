# Egg Timer
A small desktop egg timer application built with **React** and **Electron**

## Features
- Egg cooking presets (soft-boiled, medium, hard-boiled, fried)
- Visual countdown timer
- Sound notification when the timer ends

## Tech Stack
- React
- TypeScript
- Electron
- Vite
